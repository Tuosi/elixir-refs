# Stash

## Save

`git stash`

`git stash save 'foo'`

# List

`git stash list`

## Show

`git stash show`

`git stash show -p`

`git stash show stash@{0}`

## Retrieve

`git stash pop`

`git stash apply`

`git stash pop stash@{0}`

`git stash pop --index`


## Remove

`git stash drop`

`git stash drop stash@{0}`

`git stash clear`
