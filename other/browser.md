# Broswer

## Hotkeys

### Basic

cmd + [ / ] -> nav history
cmd + shift + [ / ]  -> nav tabs
cmd + w / shift + w -> close / extract tab

### Vimium

j / k -> up / down (line)
u / d -> up / down (half page)
gg / G -> top / bottom

f -> links

t -> new tab
r -> reload page
x -> close tab
X -> get it back
yy -> copy url

O -> open url, history
B -> search bookmark
T -> search tabs

i -> insert mode
v -> visual mode
esc -> back to normal mode

? -> help
