# Selectors

## Basic Selectors

1. Universal Selector

`* {}`

2. ID Selector

`#id {}`

3. Class Selector

`.class {}`

4. Type Selector

`h1, h2 ,h3 {}`

5. Attribute Selector

`div[attribute="SomeValue"] {}`

## Combinators

1. Descendant Selector

`p a {}`

2. Child Selector

`ul > li {}`

3. Adjacent Sibling Selector

`h1 + p {}`

4. General Sibling Selector

`h1 ~ p {}`
