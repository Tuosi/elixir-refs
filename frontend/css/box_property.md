# Box Properties

1. Box Sizing

    `box-sizing: border-box | content-box`

2. Margin

    `margin: 2px 4px 6px 8px | 0 auto`

3. Padding

    `padding: 2px 4px 6px 8px`

4. Border Color

    `border-color: #2AA9E0`

5. Border Style

    `border-style: none | hidden | dotted | dashed | solid | double | groove | ridge | inset | outset`

6. Border Width

    `border-width: 10px`
